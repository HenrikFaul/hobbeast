## Delivery rules
- Append-only changelog and lessons updates.
- No regressions to already working batch user generation.
- Full-context fixes only; do not hotpatch a single file in isolation.
- Prefer robust select/update-or-insert flows over fragile `upsert(... onConflict ...)` when schema drift is possible.
- Preserve UI responsiveness and accessibility.
