# 14040714_v1.4.8 AI fejlesztői promptok — Hobbeast local sync log viewer

## Üzleti kérés
A Hobbeast admin felületen a lokális címtábla batch feltöltése már elindul, de a felhasználó továbbra sem látja átláthatóan, hogy ténylegesen mi történik. Kérés: a Local Catalog szekcióban jelenjen meg egy log viewer, amelyből látható a címfeltöltés aktuális állapota és a háttérbatch eseményei.

## Elvárt fejlesztés
1. Készüljön adatbázis oldali naplótábla a lokális címszinkronhoz.
2. A `sync-local-places` edge function írjon strukturált naplóbejegyzéseket.
3. A `status` válasz adja vissza a legfrissebb naplókat is.
4. Az admin UI a Local Catalog kártyában jelenítse meg a naplót gördíthető listában.
5. A jelenlegi batch / scheduler / config funkcionalitás maradjon meg.

## Korlátok
- Nem szabad meglévő működő funkciót kivenni.
- A patch csak a valóban cserélendő fájlokat tartalmazza.
- A changelog és lessons továbbra is append-only szemléletben frissül.
