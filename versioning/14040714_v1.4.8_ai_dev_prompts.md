# 14040714_v1.4.8 AI fejlesztői promptok — Hobbeast local sync admin schema-cache bypass

## Üzleti kérés
A Hobbeast admin UI-ban a lokális címtábla menüpontok megjelentek, de a gombok továbbra is schema-cache hibákat dobálnak (`public.app_runtime_config`, `public.enqueue_local_places_batch`). A backend eközben már működik és ténylegesen tölt rekordokat a lokális táblába.

## Gyökérok
A kliensoldali admin komponens közvetlen PostgREST table/RPC hívásokat használ a lokális sync konfiguráció és batch enqueue műveletekhez, miközben a működő útvonal valójában az edge function.

## Elvárt javítás
- A browser UI ne hívja közvetlenül az új táblát és RPC-ket.
- A UI minden lokális sync műveletet a `sync-local-places` edge functionön keresztül végezzen.
- A meglévő batch sync és státusz lekérdezés maradjon működőképes.
- Az admin konfigurációs és batch gombok ne dobjanak schema-cache hibát.
