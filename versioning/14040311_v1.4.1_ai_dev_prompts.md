# Hobbeast AI fejlesztési promptok
Azonosító: `14040311`  
Verzió: `v1.4.1`

## Feladat
Valósítsd meg a Hobbeast admin felületén:
1. többprovideres eseményimport hubot az Import tabon,
2. runtime-ból állítható címkereső provider konfigurációt,
3. provider tesztelést és preview-t,
4. lokális címtábla újratöltést és állapotellenőrzést.

## Kötelező előfeltételek
- Olvasd be teljesen a `codingLessonsLearnt.md` fájlt.
- Olvasd be a `changelog.md` fájlt.
- A meglévő működő organizer / admin / events flow-kat ne rontsd el.
- A megoldás legyen patch-only szállítható.

## Megoldási elv
- Használd a meglévő Eventbrite/Ticketmaster/SeatGeek integrációs kódot, ne építs párhuzamos második implementációt.
- Az admin Import panel legyen a központi UI:
  - Eventbrite
  - Ticketmaster
  - SeatGeek
  - Címkereső / provider konfiguráció
- A címkereső provider kiválasztása legyen adatbázisban tárolt runtime config.
- Az AddressAutocomplete és a központi kereső wrapper ugyanazt a configot kövesse.
- A lokális címtábla újratöltés Edge Functionnel történjen, és adminból gombnyomással triggerelhető legyen.
